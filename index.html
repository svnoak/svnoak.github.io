Browser Plugins

What is a browser plugin?

A browser plugin is a program that you install in your browser. They can add things to your browser for example a program to manage your bookmarks or passwords.

Why do I need to use browser plugins?

You can use browser plugins to make your browser better, for example block all advertisement (so that you don’t have to see it)

Recommended browser plugins

Search Engine

What is a search engine?

The most known search engine is made by Google and you probably use it every day. A search engine (like Google) searches the internet for the words you search and shows you all it finds. Other search engines you might now are Bing or Yahoo.

Why do I need to use a different search engine?

We use a search engine we want to find the right information on the internet (or the right internet page). When you use Google, Yahoo or Bing, they save everything you search and create a profile and sell what they learn about you to other companies. This means they might now if you are sick, what your mental health state is or what you plan doing with your life. Using a different search engine can protect your privacy and doesn’t give away or save what you search online.

Recommended search engine

DuckDuckGo
